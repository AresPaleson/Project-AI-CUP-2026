# Project-AI-CUP-2026
Implements Machine learning models to win AI CUP 2026

- To install dependencies -> uv sync
- To run the models -> uv run 'file-name'
- To manage dependencies -> uv add 'package-name', uv remove 'package-name'

early-models folder consists of previous models.
